2 Cari Nanny & Mulai Matching
Direktori Nanny — /dashboard/parent/cari-nanny/direktori/
Filter: kota, tipe nanny. Tampil kartu nanny: foto, usia, pengalaman, kota
GET /api/nanny/directory?city=...&type=...
Lihat Detail Nanny (Drawer)
Profil lengkap, skor matching (jika sudah dihitung), tombol "Mulai Matching" atau "Pakai kuota"
Tes Kecocokan — /dashboard/parent/matching/survey/
53 pertanyaan · 9 subdomain (A1–C4) · pilihan ganda + skala + free text · dapat tandai DEALBREAKER
POST /api/survey/save
≥80% → teal "sangat cocok" 60–79% → orange "cukup cocok" <60% → red "perlu dipertimbangkan"
Kalkulasi Skor AI — Otomatis
Jawaban parent + nanny dibandingan → Claude API (claude-sonnet-4-20250514) → JSON: skor per domain, kekuatan, mismatch, tips → disimpan di MatchingResult (cache permanen)
POST /api/matching/score
Hasil Matching — /dashboard/parent/matching/[id]/
Skor keseluruhan 0–100, per domain A/B/C, area kekuatan & mismatch, tips, negosiasi dealbreaker
Cocok → Journey 3: Placement Tidak cocok → kembali ke direktori

Shared Flows — Lintas Role

MatchingRequest States

Subscription States

INACTIVE → bayar Mayar → ACTIVE → expired → EXPIRED

Evaluation States

PENDING → parent isi → PARENT_DONE → nanny isi → COMPLETED
