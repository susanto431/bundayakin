# **6. Sistem Matching — 3 Layer**

| **Layer** | **Nama** | **Isi** | **Harga ****&**** Output** |
| --- | --- | --- | --- |
| 1 | Survey Kecocokan | Survey paralel nanny & orang tua — diisi independen, AI matching dari dua database. Pilihan ganda praktis + bar skala + free text kondisi khusus. | GRATIS (dalam langganan 500rb). Output: % kecocokan, area match/tidak match, pertanyaan negosiasi, tips kedua pihak. |
| 2 | + Psikotes | Nanny isi tes IQ/daya tangkap, kepribadian, sikap kerja. Orang tua isi ekspektasi terstruktur. | +Rp 300rb. Output: penjabaran detail per aspek, kekuatan, gap, poin negosiasi. |
| 3 | + Psikolog | Psikolog interview nanny (PRIVAT — orang tua tidak hadir). Bisa tambah tes grafis, tes klinis. Bukan diagnostik — antisipasi risiko. | +Rp 1,2–1,5 juta. Output: catatan psikolog, pertanyaan pendalaman, saran untuk kedua pihak. |

| **Modul** | **Isi** |

## **8B. Aturan Akses**

- Akses knowledge transfer tersedia untuk orang tua yang berlangganan aktif

- Jika langganan habis dan nanny resign → berlangganan lagi, akses data kembali, minta nanny baru isi pembaruan

- Ganti nanny = trigger otomatis untuk evaluasi penutup + kompilasi knowledge package + sesi matching baru

- Nanny baru terima akses modul + onboarding checklist hal yang perlu diketahui tentang si Kecil

## **8C. Pengingat Update Database**

- Default interval: setiap 3 bulan (bisa diubah di settings profil orang tua)

- Pertanyaan pengingat: apakah ada ganti dokter/kontak darurat? kebiasaan tidur baru? makanan favorit baru? milestone baru? nanny masih sama?

- Jika nanny sudah ganti → sistem inisiasi alur ganti nanny otomatis

# **10. Fitur Platform Lengkap**

## **10A. Core Features (MVP)**

- Akun orang tua + profil anak/lansia: nama, usia, alergi, pantangan, jadwal, kondisi kesehatan

- Akun nanny + form kecocokan + rekam jejak bertahap (seperti isi CV)

- Sistem matching Layer 1: survey paralel + AI scoring + laporan kecocokan

- Dashboard orang tua: lihat laporan kecocokan, upgrade add-on, evaluasi berkala

- Payment gate: Rp 500rb/tahun aktifkan fitur penuh. Placement fee & add-on sebagai transaksi terpisah.

- T&C data nanny saat registrasi — legal foundation

## **10B. Features Post-MVP (Phase 2+)**

- Log aktivitas harian nanny — menggantikan WA group

- Evaluasi berkala otomatis (week 1, 2, bulan 1, bulan 3, setiap 3 bulan)

- Knowledge transfer saat ganti nanny: semua data anak tersimpan + pengingat update tiap 3 bulan

- Track record dua arah: orang tua beli akses review nanny · nanny beli akses review pemberi kerja

- Sosmed screening: self-serve AI, sumber komunitas WA/FB + berita publik, disclaimer indikasi awal

- Sistem referral + insentif nanny (bonus bertahan 1 & 3 bulan + badge Terpercaya)

- Tracking tumbuh kembang anak / kondisi lansia

- Video call terintegrasi — 3 skenario: nanny promosi diri / nanny-OT interview / nanny-psikolog Layer 3

- Placement harian/hourly — babysitter on-demand, target mahasiswa part-time

- Community platform
